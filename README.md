# AI CMO — 퍼포먼스 마케팅 통합 분석 대시보드 (데모)

> **▶ 라이브 데모:** https://marketing-dashboard-demo-br6gpdt4i2yrshdygxanp8.streamlit.app/
> PC가 꺼져 있어도 24시간 접속됩니다. 필터·드릴다운·탭을 직접 만져볼 수 있습니다.

Claude Code로 **직접 설계·개발**한 실제 광고 운영 대시보드의 데모 버전입니다.
표시되는 모든 수치는 **가상(임시) 데이터**이며 실제 광고주 데이터가 아닙니다. API 연동 없이 메모리에서 데이터를 생성합니다.

---

## ⭐ STAR

### S — Situation (상황)
치과 임플란트 퍼포먼스 마케팅을 **Google Ads(메인·서브)·Meta·TikTok 4개 매체**로 **혼자** 운영했습니다.
- 다매체를 혼자 다루다 보니 **매체마다 데이터 가공에 시간이 오래 걸렸고**, 매일 같은 수작업이 반복됐습니다.
- **캠페인 → 광고그룹 → 소재**까지 단계별로 파고드는 분석을 **체계적으로** 할 수 있는 도구가 필요했습니다.
- **경쟁사 분석부터 시기성(시즌·외부 이벤트) 분석**까지, 운영 의사결정을 뒷받침할 통합 대시보드가 필요했습니다.
- **실시간 성과 모니터링**이 필요한 순간에도, 4개 매체를 효율적으로 한 화면에서 다룰 수단이 없었습니다.

### T — Task (과제)
- 4채널을 **한 화면에서 통합 진단**하고,
- 캠페인의 **수명(풀 소진 시점)을 데이터로 포착**하며,
- 재세팅·소재 교체·차단 같은 액션의 **타이밍을 룰 기반으로 판단**하는 운영 시스템을 직접 구축한다.

### A — Action (실행)
Claude Code를 활용해 **대시보드 + 자동 모니터링 봇 + 경쟁사 분석 + AI 영상 파이프라인**을 직접 개발했습니다.

데모에서 체험 가능한 분석 모듈:

| 탭 | 무엇을 하는가 |
|----|--------------|
| 📊 **실시간** | 4채널 통합 KPI(전환·소진·CPM·CPC·CVR·CPA)를 한눈에 |
| 📅 **일자별** | 캠페인 → 광고그룹 → 소재 드릴다운, 기간 A/B 비교, 지표 선택 필터 |
| 🔬 **A/B테스트** | 검증 가설 단위 실험 관리 + 운영 일자별 성과 + 평가/평가 연장 |
| 🆕 **신규소재** | 신규 자산 투입 후 일자별 성과 추적 (매체별 분리) |
| 🔁 **월간 중복DB** | DB 중복률 추이 — **차단 적용 효과·풀 소진**을 월 단위로 모니터링 |
| 📺 **노출 YouTube 채널** | 디멘드젠 캠페인별 노출 채널 풀 회전 추적 |
| 🔍 **경쟁사 분석** | LP **변경 감지(SHA-256 해시 비교)** + 가격·훅·CTA 비교 |
| 📊 **코호트** | **캠페인 수명 진단** — 지표 선택(CPA·중복DB율·CVR·노출) + 3지표 겹쳐보기 |

확립한 분석 방법론(데모에 그대로 반영):
- **코호트 분석으로 캠페인 수명 측정** — 가로축을 '투입 후 경과 주차'로 정렬해 어느 시점에 풀이 마르는지 포착, 대각선으로 시즌·외부 이벤트 분리
- **무너지는 인과 사슬 추적** — `CVR↓ → 중복DB율↑ → CPA↑` 순서를 한 차트에서 확인해 풀 소진을 조기 진단
- **실 전환 기준 CPA** — API 전환값이 아닌 실제 시트 전환 기준으로 효율 판정 (API는 누락 발생)
- **보수적 OFF 룰** — 단일 일자 스파이크가 아닌 7일 평균 + 3일 연속 추세로 판단해 오판 방지

### R — Result (결과)
- **흩어진 4채널을 단일 대시보드로 통합** — 매일 수작업 집계를 자동화하고, 매체 간 채널 중복·비용 누수를 정량화
- **운영 판단을 직관에서 데이터로 전환** — 캠페인 수명·풀 소진 시점을 코호트로 가시화해 재세팅/차단 타이밍을 근거 기반으로 결정
- **잘못된 소재 OFF·리셋 오판 감소** — 룰 기반 진단으로 학습 페이즈 캠페인을 함부로 건드리는 실수 방지

> 📌 정량 성과(CPA 개선폭·내원 전환율 등)는 검증된 실측치로 채워 넣습니다.
> `[CPA __% 개선 / 월 __건 전환 / ROAS __ 달성]`
